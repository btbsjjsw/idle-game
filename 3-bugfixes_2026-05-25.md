# 2026-05-25 19:36 - 三大Bug修复

## 问题1: 神器初始解锁价格改为1点 🔮
- **位置**: `getNextUnlockPrice()` L1794
- **修改**: `Math.floor(100 * Math.pow(1.5, ...))` → `Math.floor(1 * Math.pow(1.5, ...))`
- **结果**: 第一个神器解锁只需1点，后续按1.5^n递增

## 问题2: 精英Boss能力只有文字没有效果
- **根因**: eliteAbilities数组只在updateBoss()中生成并显示文字，但攻击/防御逻辑未实现
- **修复**: 对所有10种精英能力添加真实效果

### 攻击性（Boss打玩家）- startBossAttack()
| 能力 | 效果 |
|------|------|
| eliteDouble | 每回合攻击2次 |
| eliteFire | 伤害×1.3 + 火焰特效数字 |
| eliteLightning | 伤害×1.5（叠加eliteDmgMult） |
| elitePoison | 伤害×1.1 + 玩家每秒持续掉血2%最大HP |

### 防御性（玩家打Boss）- attackBoss() / startAutoAttack()
| 能力 | 效果 |
|------|------|
| eliteDodge | 30%概率闪避 → "💨闪避!" |
| eliteDefense | 伤害×0.5（50%减免） |
| eliteShield | 伤害再×0.7（总计65%减免） |
| eliteThorns | 反弹20%伤害给玩家 |

### 持续效果
| 能力 | 效果 |
|------|------|
| eliteRegen | 每2秒回复5%最大HP (startRegen内新增eliteBossRegen间隔) |
| elitePoison | 每秒造成2%玩家最大HP的毒伤 (startPoisonTick内新增) |
