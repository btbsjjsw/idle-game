# 任务摘要：新增玩家升级系统

**时间**: 2026-05-25 14:14
**目标**: 为放置游戏新增升级回血、升级血量、主角经验条三个功能

---

## 修改内容

### 1. gameState 新增字段
- `regenLevel: 0` — 回血等级
- `hpLevel: 0` — 血量等级
- `playerExp: 0` — 当前经验
- `playerExpNeeded: 100` — 升级所需经验
- `playerLevel: 1` — 玩家等级（区别于 clickLevel）

### 2. 新增函数
- `upgradeRegen()` — 升级回血，成本公式 `15 * level^1.5`，每秒回复 `regenLevel * 5` HP
- `upgradeHP()` — 升级血量，成本公式 `20 * level^1.5`，每级增加 50 最大生命值
- `startRegen()` — 每秒回血定时器（独立 safeInterval）
- `addPlayerExp(amount)` — 添加经验值
- `checkPlayerLevelUp()` — 检查并处理玩家升级，升级后增加点击伤害

### 3. 修改函数
- `updatePlayerHP()` — 加入 `hpLevel * 50` 影响最大生命值
- `updateDisplay()` — 显示玩家等级、经验条、新升级按钮禁用状态
- `updateUpgradeCosts()` — 显示回血/血量升级成本
- `getMaxAffordable()` — 支持 'regen' 和 'hp' 类型
- `killBoss()` — 击杀时获得经验奖励 `10 + level * 2`
- `rebirth()` — 重置新字段为默认值
- `loadGame()` — 兼容旧存档，新字段默认值为0/1
- `resetAllGameData()` — 重置新字段

### 4. HTML 修改 (idle_game.html)
- 新增两个升级按钮（升级回血、升级血量）
- 新增经验条 UI（在血量条下方）

### 5. CSS 修改 (style.css)
- 新增 `.player-exp-wrap`, `.player-exp-fill`, `.player-exp-text` 样式（金色主题）

---

## 经验系统规则
- 击杀 Boss 获得经验：`10 + level * 2`
- 升级所需经验：`Math.floor(prev * 1.2 + 50)`
- 升级效果：`clickDamage += 1`, `clickLevel += 1`

## 回血系统规则
- 每秒回复：`regenLevel * 5` HP
- 仅在玩家存活且未满血时生效

## 血量系统规则
- 每级增加 50 最大生命值
- 升级时当前血量按比例增加
